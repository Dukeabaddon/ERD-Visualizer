# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2025-11-20

- Initial public preview of the ERD Visualizer extension.
- Command Palette + editor-title icon entry points.
- SQL and JSON schema parsing pipeline with normalization.
- Interactive canvas with pan/zoom/drag, relationship badges, layout persistence.
- Export popover for SVG/PNG capture with background controls.
